Uso de BrandCaptcha con ASP

La libreria en ASP de BrandCaptcha provee de manera simple colocar un BrandCaptcha en su sitio desarrollado en ASP, ayudandolo a frenar bots que intenten generar abusos del mismo. La libreria realiza comunicaciones directamente con la API de BrandCaptcha.

Después de obtener las diferentes API Keys, puede utilizar BrandCaptcha en su sitio ASP incluyendo al inicio de su página el archivo "brandcaptchalib.asp".

En el archivo "example.asp" se encuentra una implementación de ejemplo que puede utilizar como base para su sitio.
